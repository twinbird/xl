# xl
xlsx operation command line tool

## Install

```sh
go get github.com/twinbird/xl
```

## Usage

  xl -in [filepath]  
  or  
  xl -out [filepath]  

#### Option

  -S string  
        operation sheet name  
  -b string  
        base cell position  
  -in string  
        input excel file path  
  -out string  
        output excel file path  
  -s string  
        text column separater character (default " ")
